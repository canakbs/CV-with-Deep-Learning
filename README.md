# Computer Vision with Deep Learning

Hands-on implementations of core Computer Vision techniques using deep learning — from basic neural networks to object detection with YOLO.

---

## What's Inside

| Folder | Description |
|--------|-------------|
| `MNIST_ANN` | Handwritten digit classification using a basic Artificial Neural Network |
| `MNIST_GAN` | Generating handwritten digits with a Generative Adversarial Network (GAN) |
| `FLOWERS_CNN` | Flower species classification using a Convolutional Neural Network |
| `TRANSFER_LEARNING` | Image classification using pre-trained models (transfer learning) |
| `TRAFFIC_YOLO` | Real-time traffic object detection using YOLO |

## Tech Stack

`Python` `PyTorch` `TensorFlow` `OpenCV` `YOLO` `CNN` `GAN`

## Purpose

Personal practice notebooks built to develop a solid understanding of computer vision fundamentals — from classical CNNs to modern object detection pipelines.
